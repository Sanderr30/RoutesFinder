# Route Finder — Асинхронная версия

Эта ветка содержит **асинхронную реализацию** консольного приложения для поиска маршрутов между любыми двумя городами.

## Особенности

* Асинхронные HTTP-запросы: возможна параллельная обработка нескольких маршрутов.
* Более высокая производительность по сравнению с синхронной реализацией.
* Кэширование результатов (память + файловая система).


Общее описание проекта см. в [README основной ветки](../../tree/main).
